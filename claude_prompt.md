# Chirper Design System - Claude Prompt

You're helping design a unique social media app called **Chirper** (Twitter alternative) that looks great on desktop and web.

## Current Tech Stack
- Flutter with Riverpod
- Supabase (backend)
- Material 3 design system

## Current Issues
- UI looks too much like Twitter (blue bird, standard feed layout)
- Needs to look distinctive and modern
- Must work well on desktop and web (not just mobile)

## App Features
- Feed with posts, likes, comments
- User profiles with avatars
- Follow/Unfollow system
- Image uploads
- Dark mode support
- Bottom navigation

## What I Need From You

### 1. Color Palette Ideas
Suggest a fresh color scheme that's NOT the standard Twitter blue. Consider:
- Purple/Violet themes
- Coral/Sunset themes  
- Emerald/Green themes
- Or something completely unique

### 2. Layout Concepts
- Desktop: Sidebar navigation? Split view? Cards?
- Web: How should the feed look?
- Post cards: Different shapes, animations, interactions?

### 3. Typography
- Font recommendations
- Text sizes for readability on large screens

### 4. Component Ideas
- Unique button styles (not standard Material)
- Custom cards, avatars, badges
- Animations that feel premium

### 5. Desktop/Web Specific
- How to utilize wide screens
- Multi-column layouts
- Hover states, keyboard shortcuts

## Deliverables Please Provide
1. Color hex codes with names
2. CSS/Flutter widget concept descriptions
3. Layout sketches (text descriptions fine)
4. Animation ideas
5. Any reference libraries/packages that could help

---

Please generate creative, distinctive design ideas that make Chirper feel like its own platform - not a Twitter clone. Think modern tech blogs, Discord, Slack, or modern social apps like Mastodon or Bluesky's newer designs.
